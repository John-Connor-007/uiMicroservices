# uiMicroservices

Siga los siguientes pasos:

1. Hacer fork de este proyecto.
2. Clonar en el entorno local.
3. abrir command line
4. ir al directorio donde se descargo el proyecto y abrir la ruta uitest
5. correr usando el comando java -jar start.jar
6. ir a la url http://localhost:9090/ui/index.html
7. Probar
